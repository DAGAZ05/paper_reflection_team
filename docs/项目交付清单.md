## 🚀 部署建议

### 开发环境
1. Python 3.8+
2. 安装依赖: `pip install -r requirements.txt`
3. 设置环境变量: `DEEPSEEK_API_KEY`
4. 运行测试: `python tests/generate_test_data.py`

### 生产环境
1. 配置数据库连接（10.13.1.26）
2. 确保paper_sections表有数据
3. 配置DeepSeek API密钥
4. 使用数据库模式运行: `python run.py --mode database`
